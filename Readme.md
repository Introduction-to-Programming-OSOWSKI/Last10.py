# **Last 10 **  

## **Assignment Description**  
Create a function called last10() that returns True if the last value of a given list a is equal to 10. Return False if the last value is not 10.

## **Examples**  
>last10([2,2,2,2,2])  
False  

>last10([2,2,2,2,10])  
True  

>last10([45,345,22,117,10])  
True
